مهارت ها

با بیش از 6 سال تجربه در بازار کار و انواع پروژه های شخصی, شرکتی و فروشگاهی توانایی هایی کد زبان های HTML, css, js, PHP کسب کردم.

HTML

این زبان نشانه‌گذاری برای طراحی ساختار صفحه‌های اینترنتی به کار می‌رود. با استفاده از آن می‌توان بخش‌های مختلفی مانند متن، تصویر، پیوند و جدول را در یک صفحه قرار داد. بدون این زبان، هیچ صفحه‌ای در اینترنت نمایش داده نخواهد شد.

CSS

این زبان برای طراحی ظاهر صفحه‌های اینترنتی به کار می‌رود. با کمک آن می‌توان رنگ‌بندی، اندازه نوشته‌ها، فاصله‌ها و نحوه‌ی چینش بخش‌های مختلف را تنظیم کرد. همچنین امکان ایجاد جلوه‌های ویژه و طراحی واکنش‌گرا را فراهم می‌کند.

JavaScript

این زبان برنامه‌نویسی برای پویاسازی صفحه‌های اینترنتی استفاده می‌شود. با کمک آن می‌توان دکمه‌های تعاملی، انیمیشن‌ها و تغییرات لحظه‌ای در صفحه ایجاد کرد. همچنین برای پردازش داده‌ها و ارتباط با سرور کاربرد دارد.

PHP

این زبان برنامه‌نویسی برای ساخت بخش‌های پنهان وب‌سایت‌ها به کار می‌رود. با استفاده از آن می‌توان اطلاعات را در پایگاه داده ذخیره کرد، درخواست‌های کاربران را پردازش کرد و بخش‌های گوناگون سایت را مدیریت نمود. همچنین امنیت هم دارد
